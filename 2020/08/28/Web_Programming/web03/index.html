태그 구성요소

head 태그에는 웹페이지에 대한 정보를 담는다. 페이지 인코딩 방식이나 css 파일 링크, 페이지의 제목 등 다양한 데이터를 포함한다.

meta 태그는 웹페이지에 대한 메타데이터를 나타내는 태그이다. 메타데이터란 데이터를 설명하는 데이터이다. 문서 작성자나 문서 요약과 같은 정보를 담고 있다.

주로 인코딩 방식을 나타내기 위해 사용된다.

MDN-link
link는 외부 리소스를 불러올 때 사용한다. 주로 stylesheet를 연결할 때 가장 많이 사용하지만, 다른 속성을 이용해 그 외의 resource들도 불러 올 수 있다.
